É muito conteúdo para pouco tempo, portanto não deu pra passar tudo em uma aula.
Logo, peço que assistam, por favor, a essas aulas abaixo. 
Esse conhecimento será necessário para que possamos construir o sistema final.
Favor priorizar esses assuntos e deixar os exercícios para depois (caso já tenha adquirido
familiaridade com a linguagem).

OBS 1: Deixei várias opções de vídeo, porém só precisa assistir
um dos vídeos de cada assunto (o vídeo que achar melhor)

OBS 2: Não precisa fazer os exercícios de tuple nem de set. Faça apenas
os exercícios de list, dict e arquivo, além dos outros mais simples até que você esteja
confortável com a linguagem. O objetivo não é aprendermos os detalhes da
linguagem (até porque isso leva anos ...), e sim a sintaxe, porque a lógica de
programação vocês já sabem.
